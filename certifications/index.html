| Certifications |

“A patient pursuit of facts, and cautious combination and comparison of them, is the drudgery to which man is subjected by his Maker, if he wishes to attain sure knowledge.” ― Thomas Jefferson.

Google Data Analytics Professional Certificate

Those who earn the Google Data Analytics Certificate have completed eight courses, developed by Google, that include hands-on, practice-based assessments and are designed to prepare them for introductory-level roles in Data Analytics. They are competent in tools and platforms including spreadsheets, SQL, Tableau, and R. They know how to prepare, process, analyze, and share data for thoughtful action.

Data Science Fundamentals Specialization

This specialization introduces learners to the field of data science and covers essential skills, techniques, and concepts. It starts by explaining foundational concepts like analytics taxonomy and the Cross-Industry Standard Process for Data Mining, and progresses to compare data science with classical statistical techniques. Learners will also gain an understanding of various data science techniques such as data analysis, statistical modeling, big data manipulation, data mining algorithms, and data quality management.

Data Visualization with Tableau Specialization

This specialization, in collaboration with Tableau, teaches newcomers how to use data visualization effectively. Participants will learn best practices, view real-world examples, and gain skills in generating powerful reports and dashboards using Tableau, enabling them to make informed decisions based on business data. The program concludes with a Capstone Project where participants create visualizations and data models for a fictional company’s executive leadership presentation.

SQL for Data Science

This course provides a primer on SQL and data analysis for aspiring data scientists. By learning SQL, participants will acquire the skills to retrieve and manipulate data, write queries, create tables, combine data, and apply data governance and profiling techniques. This foundational knowledge will enable them to ask the right questions, deliver valuable insights, and shape data for targeted analysis purposes, ultimately preparing them for a career in data science.

Distributed Computing with Spark SQL

This course is designed for students with SQL experience who want to learn distributed computing with Apache Spark for big data analysis. The skills learned include understanding Spark architecture, performing queries in Spark, optimizing Spark SQL, and building reliable data pipelines. The course covers topics such as Spark basics, troubleshooting performance, engineering data pipelines, and working with data lakes and warehouses, enabling students to enhance their SQL and distributed computing skills for advanced analysis and transition into more advanced analytics as Data Scientists.

Data Science: Foundations using R Specialization

The Data Science Specialization covers foundational concepts and tools for the data science pipeline. Successful participants learn how to use the tools of the trade, think analytically about complex problems, manage large data sets, create visualizations, and publish reproducible analyses.

Statistical Inference

Statistical inference is the process of drawing conclusions about populations or scientific truths from data. This course presents the fundamentals of inference in a practical approach for getting things done. After taking this course, students will understand the broad directions of statistical inference and use this information for making informed choices in analyzing data.

Practical Machine Learning

This course will cover the basic components of building and applying prediction functions with an emphasis on practical applications. The course will provide basic grounding in concepts such as training and tests sets, overfitting, and error rates. The course will also introduce a range of model based and algorithmic machine learning methods including regression, classification trees, Naive Bayes, and random forests. The course will cover the complete process of building prediction functions including data collection, feature creation, algorithms, and evaluation.

Regression Models

Regression models, a subset of linear models, are the most important statistical analysis tool in a data scientist’s toolkit. This course covers regression analysis, least squares and inference using regression models. Special cases of the regression model, ANOVA and ANCOVA will be covered as well. Analysis of residuals and variability will be investigated. The course will cover modern thinking on model selection and novel uses of regression models including scatterplot smoothing.

Python for Everybody Specialization

This Specialization builds on the success of the Python for Everybody course and will introduce fundamental programming concepts including data structures, networked application program interfaces, and databases, using the Python programming language. In the Capstone Project, you’ll use the technologies learned throughout the Specialization to design and create your own applications for data retrieval, processing, and visualization.

Statistics with Python Specialization

This specialization is designed to teach learners beginning and intermediate concepts of statistical analysis, and use of the Python programming language to conduct data analyses. Learners will learn where data come from, what types of data can be collected, how to effectively summarize and visualize data, how to utilize data for estimation and assessing theories, proper interpretations of inferential results, and how to apply more advanced statistical modeling procedures.

Applied Data Science with Python Specialization

The 5 courses in this University of Michigan specialization introduce learners to data science through the python programming language. This skills-based specialization is intended for learners who have a basic python or programming background, and want to apply statistical, machine learning, information visualization, text analysis, and social network analysis techniques through popular python toolkits such as pandas, matplotlib, scikit-learn, nltk, and networkx to gain insight into their data.

Data Science Ethics

This course explores the ethical considerations surrounding consumer privacy and control of big data in the wake of recent data breaches. It covers topics such as the broader societal impact of data science, principles of fairness and transparency, responsible data management, and the legal aspects of privacy rights and voluntary disclosure. The course is suitable for data scientists and individuals looking to utilize or expand their use of data, requiring no specific prior knowledge.

Machine Learning Specialization

This Specialization from leading researchers at the University of Washington introduces you to the exciting, high-demand field of Machine Learning. Through a series of practical case studies, you will gain applied experience in major areas of Machine Learning including Prediction, Classification, Clustering, and Information Retrieval. You will learn to analyze large and complex datasets, create systems that adapt and improve over time, and build intelligent applications that can make predictions from data.

TensorFlow 2 for Deep Learning Specialization

This Specialization in TensorFlow is designed for machine learning researchers and practitioners aiming to acquire practical skills. The first course covers fundamental concepts, including model building, training, evaluation, prediction, regularization, callbacks, and saving/loading models. The second course focuses on developing customized deep learning models, utilizing lower-level APIs, complex architectures, custom layers, and sequence models. The final course specializes in probabilistic approaches, teaching how to develop probabilistic models using TensorFlow Probability, bridging probabilistic models with deep learning. Prerequisite skills include Python 3, general machine learning and deep learning knowledge, and a solid foundation in probability and statistics.

Mathematics for Machine Learning: Linear Algebra

This course on Linear Algebra explores the concepts of vectors and matrices, including eigenvalues and eigenvectors, and their applications in solving problems and working with datasets. Participants will learn how to rotate images, extract eigenvectors, and understand the Pagerank algorithm. The course also includes hands-on coding exercises using Jupyter notebooks in Python, aimed at providing a practical understanding of the concepts and their application in machine learning.

Mathematics for Machine Learning: Multivariate Calculus

This course provides a brief introduction to multivariate calculus for building machine learning techniques. Skills learned include understanding the gradient of a function, calculating vectors on multidimensional surfaces, building function approximations, applying calculus in neural network training, and using it in linear regression models. The course aims to develop an intuitive understanding of calculus and provide the necessary language for self-guided learning, enabling students to pursue more advanced machine learning courses with confidence.

Modern Big Data Analysis with SQL Specialization

This Specialization teaches essential skills for working with large-scale data using SQL. It focuses on gaining the skills necessary to query big data with modern distributed SQL engines like Hive and Impala. The courses cover the basics of SQL as well as the specific techniques required to query huge datasets stored in distributed clusters and cloud storage using these advanced SQL engines.

Databases for Data Scientists Specialization

This specialization is designed for beginning programmers, data scientists, and software developers who want to learn about relational databases. The skills that will be learned include database design, fundamentals of database software, and using SQL to work with databases. The specialization also provides an overview of future trends in databases such as NoSQL and Big Data.

Excel Skills for Data Analytics and Visualization Specialization

This specialization focuses on developing analytical and visualization skills, specifically in advanced Excel functions and beginner Power BI. The skills learned include data transformation, analysis, linking, and visualization through charts and interactive dashboards. Additionally, the course covers advanced Excel features like PowerPivot, Get and Transform, and DAX, which allow for automated data transformation, removal of row limitations, and integration with Power Business Intelligence.

Version Control with Git

This course offers a hands-on foundation for understanding Git, a widely used version control system for managing team files and improving products. The course assumes no prior Git knowledge and provides a strong conceptual understanding, allowing learners to delve deeper into specific topics. The course offers two paths, one using the command line interface and the other utilizing the Sourcetree graphical client, with hands-on labs requiring a Windows, Mac, or Linux computer. The skills learned include Git version control, team collaboration, DevOps, continuous delivery pipelines, and cloud-native computing.

Practical Time Series Analysis

This course is designed for individuals with technical competencies who lack formal analytic training but want to enhance their understanding of routine presentation and analysis. The course focuses on practical Time Series Analysis, exploring mathematical models and graphical representations for sequential data sets. Participants will also learn forecasting techniques for future predictions. The course utilizes R, a free implementation of the S language, and offers video lectures, written materials, quizzes, and opportunities for discussion among learners. Key skills learned include practical time series analysis, mathematical modeling, graphical representation, and forecasting.

Data Science Math Skills

This data science math course teaches learners the fundamental math skills necessary for success in data science. Topics covered include set theory, properties of real numbers, algebraic concepts, Cartesian plane math, functions and their inverses, exponential and logarithmic functions, and probability theory. Completion of this course equips learners with the vocabulary, notation, and algebra rules essential for further advanced data science courses, including “Mastering Data Analysis in Excel.”

Apache Spark (TM) SQL for Data Analysts

This course focuses on teaching students how to utilize Apache Spark for big data analytics by leveraging their existing SQL skills. They will learn how to work with Delta Lake, an open-source storage layer that enhances data lake reliability. By the end of the course, students will gain proficiency in using Spark SQL and Delta Lake to ingest, transform, and query data for valuable insights and collaborative sharing. Skills learned include Apache Spark, Spark SQL, and Delta Lake.

Microsoft Azure Machine Learning for Data Scientists

This course focuses on using Azure Machine Learning to create and publish models without coding. The skills learned include managing Azure resources for machine learning, running experiments and training models, deploying and operationalizing machine learning solutions, implementing responsible machine learning, and integrating Azure Databricks with Azure Machine Learning. This course is part of a program preparing students for the DP-100 certification exam, which assesses knowledge and expertise in operating machine learning solutions at a cloud-scale using Azure Machine Learning.

Data-Driven Decisions with Power BI

This course is designed for new Power BI users to gain a conceptual understanding of the Power BI desktop application and service. They will learn to manage pages, create visualizations, import, connect, clean, transform, and model their own data. Additionally, they will explore reports, workspaces, and become proficient in creating and utilizing dashboards. The skills learned include conceptual understanding of Power BI, data management, visualization creation, report exploration, and dashboard creation.

Neural Networks and Random Forests

This course focuses on advancing AI techniques through a deep dive into neural networks and coding simple models, emphasizing avoiding overfitting and regularization. The skills that will be learned include understanding neural network structure and properties, implementing neural network models, and applying hyper-parameter tricks. Additionally, the course covers random forests, highlighting the differences between neural networks and random forests and completing a project on predicting similarity between health patients using random forests.

Demand Forecasting Using Time Series

This course covers various aspects of time series, with a specific focus on demand prediction. Skills that will be learned include understanding concepts like stationarity, trend, cyclicality, and seasonality in time series analysis, as well as analyzing correlation methods, particularly autocorrelation. Additionally, participants will learn methods for demand prediction using time series, including autoregressive models, and will have an opportunity to apply their knowledge by completing a project that involves predicting demand using ARIMA models in Python.

Microsoft Power BI Desktop for Business Intelligence

This course offers a comprehensive guide to learning Microsoft Power BI Desktop, the world’s leading self-service business intelligence platform. Through a hands-on project, you will learn to connect and transform raw data, build a relational data model, add calculated fields using DAX, and design interactive Power BI reports. Skills learned include data connection and transformation, relational data modeling, DAX calculations, and report design and visualization using Power BI.

Advanced DAX for Microsoft Power BI Desktop

This hands-on, project-based course is designed to take users’ Power BI skills to the next level by mastering Data Analysis Expressions (DAX). Skills learned include analyzing product inventory, repeat customers, store-level performance, and more using Power BI and DAX. The course covers DAX formula and storage engines, advanced time intelligence, calculated table joins, relationship functions, performance tuning, and DAX query optimization using tools like DAX Studio and Power BI’s Performance Analyzer.
